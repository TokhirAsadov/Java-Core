# Java Dasturlash Tili Haqida To‘liq Ma’lumot

Java — dunyodagi eng mashhur va ko‘p qirrali dasturlash tillaridan biri bo‘lib, Sun Microsystems tomonidan 1995-yilda ishlab chiqilgan va hozirda Oracle korporatsiyasi tomonidan boshqariladi. Java platformadan mustaqilligi, xavfsizligi va keng qamrovli kutubxonalari tufayli turli sohalarda, jumladan desktop, veb, mobil va korporativ ilovalarda keng qo‘llaniladi. Quyida Java haqida batafsil ma’lumot, uning tarixi, xususiyatlari, nashrlari, platformasi va foydalanish sohalari keltirilgan.

---

## Java Nima?

Java — ob’ektga yo‘naltirilgan, platformadan mustaqil, umumiy maqsadli dasturlash tili. U "Write Once, Run Anywhere" (WORA) prinsipi asosida ishlaydi, ya’ni bir marta yozilgan Java kodi turli operatsion tizimlar va qurilmalarda (Windows, Linux, macOS, Android va boshqalar) o‘zgartirishsiz ishlay oladi. Bu Java Virtual Machine (JVM) tufayli amalga oshiriladi, u Java kodini platformaga mos bayt-kodga aylantiradi.

Java dasturlari odatda quyidagi sohalarda ishlatiladi:
- **Desktop ilovalar**: IDE’lar (masalan, Eclipse, IntelliJ IDEA).
- **Veb-ilovalar**: Bank tizimlari, onlayn do‘konlar.
- **Mobil ilovalar**: Android dasturlari.
- **Korporativ tizimlar**: ERP, CRM va ma’lumotlar bazasi ilovalari.
- **O‘yinlar**: Minecraft kabi mashhur o‘yinlar.

---

## Java’ning Tarixi

- **1991-yil**: Sun Microsystems’da James Gosling boshchiligidagi jamoa “Oak” nomli loyiha ustida ishlay boshladi. Bu loyiha o‘rnatilgan qurilmalar uchun mo‘ljallangan edi.
- **1995-yil**: “Oak” loyihasi “Java” deb o‘zgartirildi (Indoneziyaning Java orolidan ilhomlangan) va rasman chiqarildi.
- **1996-yil**: Java 1.0 chiqdi, bu dasturlash tilining ilk versiyasi edi.
- **2006-yil**: Java ochiq manbali sifatida OpenJDK loyihasi sifatida chiqarildi.
- **2010-yil**: Oracle Sun Microsystems’ni sotib oldi va Java’ning rivojlanishini boshqardi.
- **2025-yil holatiga ko‘ra**: Java SE 21 (LTS) va undan keyingi qisqa muddatli versiyalar (masalan, Java 23) eng so‘nggi versiyalar hisoblanadi.

## Java’ning Asosiy Xususiyatlari

Java’ning mashhurligi uning quyidagi xususiyatlari bilan bog‘liq:

1. **Platformadan Mustaqillik**:
   - Java kodi JVM orqali har qanday platformada ishlaydi.
   - Misol: Bitta Java ilovasi Windows’da yozilib, Linux yoki Android’da ishlatilishi mumkin.

2. **Ob’ektga Yo‘naltirilgan Dasturlash**:
   - Inkapsulyatsiya, meros, polimorfizm va abstraksiya tamoyillari qo‘llaniladi.
   - Misol: `class Car {}` sinfi ichida `drive()` metodi inkapsulyatsiya qilinadi.

3. **Oddiylik**:
   - Java C++ kabi tillarga nisbatan soddaroq sintaksisga ega.
   - Ko‘rsatgichlar (pointers) va xotira boshqaruvi olib tashlangan.

4. **Xavfsizlik**:
   - Bytecode tekshiruvi, Sandbox modeli va Security Manager xavfsizlikni ta’minlaydi.
   - Misol: Java applets brauzerlarda xavfsiz ishlaydi.

5. **Mustahkamlik**:
   - Avtomatik xotira boshqaruvi (Garbage Collection) va exception handling xatolarni kamaytiradi.
   - Misol: `try-catch` bloklari orqali `NullPointerException` boshqariladi.

6. **Yuqori Unumdorlik**:
   - Just-In-Time (JIT) kompilyatori kodni optimallashtiradi.
   - Misol: Apache Tomcat serveri yuqori unumdorlikka ega.

7. **Ko‘p Oqimli (Multithreaded)**:
   - Bir nechta vazifalarni parallel bajarish imkoniyati.
   - Misol: Veb-serverlar ko‘p foydalanuvchi so‘rovlarini boshqaradi.

8. **Taqsimlangan Dasturlash**:
   - RMI va JMS kabi vositalar taqsimlangan tizimlarni qurishni osonlashtiradi.
   - Misol: Bulutli xizmatlar va bank tizimlari.

9. **Dinamik va Kengaytiriladigan**:
   - ClassLoader va Reflection API dasturni ish vaqtida kengaytirish imkonini beradi.
   - Misol: Eclipse plaginlarni dinamik yuklaydi.

10. **Boy API va Kutubxonalar**:
    - `java.lang`, `java.util`, `java.io`, `javax.swing`, `java.sql` kabi kutubxonalar.
    - Misol: JDBC orqali MySQL bilan ishlash.

11. **Portativlik**:
    - Standartlashtirilgan ma’lumot turlari va JVM tufayli portativ.
    - Misol: Android ilovasi turli telefonlarda ishlaydi.

12. **Katta Hamjamiyat**:
    - OpenJDK, forumlar va uchinchi tomon kutubxonalari (Spring, Hibernate).
    - Misol: Stack Overflow’da millionlab Java savollari.

## Java Platformasi

Java platformasi — dasturlarni ishlab chiqish va ishga tushirish uchun zarur bo‘lgan muhit. U quyidagi asosiy komponentlardan iborat:

1. **Java Virtual Machine (JVM)**:
   - Bayt-kodni mashina tiliga aylantiradi.
   - Garbage Collection va JIT kompilyatori unumdorlikni oshiradi.

2. **Java Runtime Environment (JRE)**:
   - JVM va standart kutubxonalar to‘plami.
   - Ilovalarni faqat ishga tushirish uchun ishlatiladi.

3. **Java Development Kit (JDK)**:
   - JRE, kompilyator (`javac`), debugger va boshqa vositalarni o‘z ichiga oladi.
   - OpenJDK — ochiq manbali muqobil.

4. **API’lar va Kutubxonalar**:
   - I/O, tarmoq, GUI (Swing, JavaFX), ma’lumotlar bazasi va boshqa funksiyalar.

---

## Java Nashrlari (Editions)

Java platformasi turli ehtiyojlarni qondirish uchun uchta asosiy nashrga bo‘linadi:

1. **Java SE (Standard Edition)**:
   - Umumiy dasturlash uchun (desktop, konsol, kichik serverlar).
   - Komponentlar: JDK, JRE, JVM, Swing, JavaFX.
   - Versiyalar: Java SE 21 (LTS, 2023) va keyingi versiyalar.

2. **Java EE (Enterprise Edition) / Jakarta EE**:
   - Korporativ ilovalar uchun (veb, server, API’lar).
   - Komponentlar: Servlets, JSP, EJB, JPA, JMS.
   - Versiyalar: Jakarta EE 10 (2025-yil holatiga ko‘ra).

3. **Java ME (Micro Edition)**:
   - Resurslari cheklangan qurilmalar uchun (eski mobil telefonlar, IoT).
   - Komponentlar: CLDC, CDC, MIDP.
   - Eslatma: Android va iOS tufayli eskirgan.

4. **Boshqa Nashrlar**:
   - **Java Card**: Aqlli kartalar uchun.
   - **JavaFX**: Zamonaviy GUI ishlab chiqish.
