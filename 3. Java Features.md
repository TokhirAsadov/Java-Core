# Java Dasturlash Tilining Xususiyatlari Haqida To‘liq Ma’lumot

Java — Sun Microsystems tomonidan 1995-yilda ishlab chiqilgan va hozirda Oracle tomonidan boshqariladigan ko‘p qirrali, ob’ektga yo‘naltirilgan dasturlash tili. Java’ning mashhurligi uning platformadan mustaqilligi, xavfsizligi va keng qamrovli xususiyatlari bilan bog‘liq. Quyida Java’ning asosiy xususiyatlari batafsil tasvirlanadi.

---

## 1. **Platformadan Mustaqillik (Platform Independence)**

- **Tavsif**: Java’ning eng muhim xususiyati — "Write Once, Run Anywhere" (WORA) prinsipi. Java kodi bir marta yoziladi va har qanday platformada (Windows, Linux, macOS, va boshqalar) o‘zgartirishsiz ishlay oladi.
- **Qanday ishlaydi**:
  - Java kodi avval **bayt-kod**ga kompilyatsiya qilinadi (`.class` fayllari).
  - **Java Virtual Machine (JVM)** bu bayt-kodni har bir platformaga mos ravishda mashina tiliga aylantiradi.
- **Afzallik**:
  - Dasturchilar har bir operatsion tizim uchun alohida kod yozishdan xalos bo‘ladi.
  - Ilovalar portativ va moslashuvchan bo‘ladi.
- **Misol**: Bitta Java ilovasi Windows’da yozilib, Linux yoki mobil qurilmalarda ishlatilishi mumkin.

---

## 2. **Ob’ektga Yo‘naltirilgan Dasturlash (Object-Oriented Programming)**

- **Tavsif**: Java to‘liq ob’ektga yo‘naltirilgan dasturlash tilidir, bu kodni modulli, qayta ishlatiladigan va boshqarish oson qiladi.
- **Asosiy tamoyillar**:
  - **Inkapsulyatsiya (Encapsulation)**: Ma’lumotlar va ular bilan ishlovchi metodlarni bir sinf ichida birlashtirish.
  - **Meros (Inheritance)**: Bir sinfdan boshqa sinfga xususiyatlar va metodlarni o‘tkazish.
  - **Polimorfizm (Polymorphism)**: Bir xil metodning turli shakllarda ishlatilishi.
  - **Abstraksiya (Abstraction)**: Murakkab tafsilotlarni yashirish va faqat muhim ma’lumotlarni ko‘rsatish.
- **Afzallik**:
  - Kodni qayta ishlatish osonlashadi.
  - Loyihalarni boshqarish va kengaytirish soddalashadi.
- **Misol**: `class Car` sinfi yaratilib, uning ichida `drive()` va `stop()` metodlari inkapsulyatsiya qilinadi.

---

## 3. **Oddiylik (Simplicity)**

- **Tavsif**: Java boshqa tillarga (masalan, C++) nisbatan soddaroq sintaksisga ega, bu uni o‘rganishni osonlashtiradi.
- **Xususiyatlar**:
  - Ko‘rsatgichlar (pointers) va xotira boshqaruvi kabi murakkab tushunchalar olib tashlangan.
  - Avtomatik xotira boshqaruvi (Garbage Collection) tufayli dasturchi xotira bilan ishlashdan xavotirlanmaydi.
- **Afzallik**: Yangi boshlanuvchilar uchun oson o‘rganiladi va xatolarni kamaytiradi.
- **Misol**: C++’dagi ko‘rsatgichlar o‘rniga Java’da oddiy obyektlarga murojaat qilish ishlatiladi.

---

## 4. **Xavfsizlik (Security)**

- **Tavsif**: Java xavfsiz dasturlar ishlab chiqish uchun mo‘ljallangan, bu uni veb va korporativ ilovalarda mashhur qiladi.
- **Xavfsizlik xususiyatlari**:
  - **Bytecode Verification**: JVM bayt-kodni ishga tushirishdan oldin tekshiradi.
  - **Sandbox Security Model**: Ilovalar cheklangan muhitda ishlaydi, bu zararli kodning tizimga ta’sirini oldini oladi.
  - **Security Manager**: Ilovaning tizim resurslariga kirishini nazorat qiladi.
- **Afzallik**: Viruslar va xavfsizlik xavflariga qarshi mustahkam himoya.
- **Misol**: Java applets brauzerlarda xavfsiz ishlaydi.

---

## 5. **Mustahkamlik (Robustness)**

- **Tavsif**: Java xatolarni oldini olish va barqaror ilovalar yaratishga yordam beradi.
- **Xususiyatlar**:
  - **Avtomatik Xotira Boshqaruvi (Garbage Collection)**: Foydalanilmayotgan obyektlar avtomatik ravishda xotiradan tozalanadi.
  - **Exception Handling**: Xatolarni boshqarish uchun try-catch bloklari.
  - **Type Checking**: Kompilyatsiya va ishga tushirish vaqtida ma’lumot turlari tekshiriladi.
- **Afzallik**: Tizimning ishdan chiqish ehtimoli kamayadi.
- **Misol**: `NullPointerException` kabi xatolarni try-catch yordamida boshqarish.

---

## 6. **Yuqori Unumdorlik (High Performance)**

- **Tavsif**: Java boshqa tillarga nisbatan sekinroq deb hisoblansa-da, **Just-In-Time (JIT) Compiler** tufayli yuqori unumdorlikka erishadi.
- **Qanday ishlaydi**:
  - JIT kompilyatori bayt-kodni mashina tiliga aylantirib, ishga tushirishni tezlashtiradi.
  - JVM optimallashtirish algoritmlari kodni samarali bajaradi.
- **Afzallik**: Katta loyihalarda barqaror va tez ishlash.
- **Misol**: Katta server ilovalari (masalan, Apache Tomcat) yuqori unumdorlikni ta’minlaydi.

---

## 7. **Ko‘p Oqimli (Multithreaded)**

- **Tavsif**: Java bir vaqtning o‘zida bir nechta vazifalarni bajarish imkonini beruvchi ko‘p oqimli dasturlashni qo‘llab-quvvatlaydi.
- **Xususiyatlar**:
  - **Thread sinfi va Runnable interfeysi**: Parallel ishlov berish uchun.
  - **Sinxronizatsiya**: Bir nechta oqimlar o‘rtasida ma’lumot almashishni boshqarish.
- **Afzallik**: Ilovalarni tezroq va samaraliroq qiladi, ayniqsa server ilovalarida.
- **Misol**: Veb-serverlar bir vaqtning o‘zida ko‘plab foydalanuvchi so‘rovlarini boshqaradi.

---

## 8. **Taqsimlangan Dasturlash (Distributed Computing)**

- **Tavsif**: Java tarmoq orqali taqsimlangan tizimlarni qurishni osonlashtiradi.
- **Xususiyatlar**:
  - **RMI (Remote Method Invocation)**: Masofadan metodlarni chaqirish.
  - **Java EE va JMS**: Taqsimlangan ilovalar va xabar almashish.
- **Afzallik**: Katta tarmoq ilovalarini (masalan, bulutli xizmatlar) ishlab chiqish oson.
- **Misol**: Bank tizimlari va onlayn xizmatlar.

---

## 9. **Dinamik va Kengaytiriladigan (Dynamic and Extensible)**

- **Tavsif**: Java dastur ishlayotgan paytda yangi sinflar va kutubxonalarni yuklash imkonini beradi.
- **Xususiyatlar**:
  - **ClassLoader**: Yangi sinflarni dinamik ravishda yuklash.
  - **Reflection API**: Kodning tuzilishini ish vaqtida tahlil qilish.
- **Afzallik**: Plaginlar va kengaytmalar yaratish oson.
- **Misol**: Eclipse kabi IDE’lar plaginlarni dinamik yuklaydi.

---

## 10. **Boy API va Kutubxonalar (Rich API)**

- **Tavsif**: Java keng qamrovli standart kutubxonalar (Java Class Library) bilan ta’minlaydi.
- **Asosiy API’lar**:
  - **java.lang**: Asosiy sinflar (String, Math).
  - **java.util**: Ma’lumotlar tuzilmalari (List, Map).
  - **java.io va java.nio**: Fayl va tarmoq operatsiyalari.
  - **javax.swing va JavaFX**: Grafik interfeyslar.
  - **java.sql va JPA**: Ma’lumotlar bazasi bilan ishlash.
- **Afzallik**: Dasturchilar ko‘p funksiyalarni noldan yozmasdan ishlatishi mumkin.
- **Misol**: JDBC yordamida MySQL bilan ishlash.

---

## 11. **Portativlik (Portability)**

- **Tavsif**: Java kodi va JVM turli apparat va operatsion tizimlarda ishlashi tufayli portativdir.
- **Xususiyatlar**:
  - Standartlashtirilgan ma’lumot turlari (masalan, `int` har doim 32 bit).
  - JVM tufayli apparatga bog‘liqlik yo‘q.
- **Afzallik**: Kodni turli qurilmalarga oson ko‘chirish mumkin.
- **Misol**: Android ilovasi Java’da yozilib, turli telefonlarda ishlaydi.

---

## 12. **Katta Hamjamiyat va Qo‘llab-quvvatlash**

- **Tavsif**: Java dunyodagi eng katta dasturchilar hamjamiyatlaridan biriga ega.
- **Xususiyatlar**:
  - Ochiq manbali OpenJDK loyihasi.
  - Forumlar, hujjatlar va darsliklarning ko‘pligi.
  - Ko‘plab uchinchi tomon kutubxonalari (masalan, Spring, Hibernate).
- **Afzallik**: Muammolarni hal qilish va yangi vositalarni topish oson.
- **Misol**: Stack Overflow’da Java bilan bog‘liq millionlab savollar.

---

## Xulosa

Java’ning xususiyatlari uni ko‘p qirrali, xavfsiz va mustahkam dasturlash tili qiladi. Platformadan mustaqillik, ob’ektga yo‘naltirilganlik, boy API’lar va katta hamjamiyat Java’ni desktop, veb, mobil va korporativ ilovalar uchun ideal tanlovga aylantiradi. Garchi ba’zi sohalarda (masalan, mobil dasturlashda) Kotlin kabi muqobillar paydo bo‘lsa-da, Java o‘zining barqarorligi va keng qo‘llanilishi tufayli hali ham yetakchi tillardan biridir.

> **Eslatma**: Agar Java’ning ma’lum bir xususiyati bo‘yicha misollar, kod namunasi yoki chuqurroq tushuntirish kerak bo‘lsa, iltimos, aniq so‘rov yuboring!
