# Windows’da Java (JDK) O‘rnatish Bo‘yicha To‘liq Yo‘riqnoma

Java Development Kit (JDK) — Java dasturlarini ishlab chiqish, kompilyatsiya qilish va ishga tushirish uchun zarur bo‘lgan asosiy vosita to‘plami. Ushbu yo‘riqnoma Windows 10 yoki 11 operatsion tizimida JDK ni o‘rnatish, muhit o‘zgaruvchilarini sozlash va o‘rnatishni sinovdan o‘tkazishni bosqichma-bosqich tushuntiradi. Quyida batafsil qadamlar keltirilgan.

---

## Talablar

Java’ni o‘rnatishdan oldin quyidagi shartlarni ta’minlang:
- **Operatsion tizim**: Windows 10 yoki 11 (64-bit tavsiya qilinadi).
- **RAM**: Kamida 512 MB (1 GB yoki undan ko‘p tavsiya qilinadi).
- **Diskdagi bo‘sh joy**: JDK uchun kamida 300 MB.
- **Internet ulanishi**: JDK yuklab olish uchun.
- **Administrator huquqlari**: O‘rnatish va muhit o‘zgaruvchilarini sozlash uchun zarur.
- **Eslatma**: Agar eski Java versiyalari o‘rnatilgan bo‘lsa, ularni o‘chirish tavsiya qilinadi, chunki bir nechta versiyalar konfliktlarga olib kelishi mumkin.[](https://phoenixnap.com/kb/install-java-windows)

---

## 1-qadam: Java’ni Yuklab Olish

JDK ni yuklab olish uchun quyidagi manbalardan birini tanlashingiz mumkin:
- **Oracle JDK**: Rasmiy Oracle veb-saytidan yuklab olinadi[](https://www.oracle.com/java/technologies/downloads/). Kommersial foydalanish uchun litsenziya talab qilinishi mumkin.
- **OpenJDK**: Ochiq manbali va bepul muqobil (https://adoptium.net/ yoki https://openjdk.java.net/). Yangi boshlanuvchilar uchun tavsiya qilinadi, chunki litsenziya cheklovlari kamroq.[](https://www.theserverside.com/blog/Coffee-Talk-Java-News-Stories-and-Opinions/How-do-I-install-Java-on-Windows)

### Yuklab olish jarayoni:
1. Veb-brauzerni oching va quyidagi saytlardan biriga o‘ting:
   - Oracle JDK uchun: https://www.oracle.com/java/technologies/downloads/
   - OpenJDK uchun: https://adoptium.net/
2. Operatsion tizimingizga mos versiyani tanlang:
   - 64-bit Windows uchun “x64 Installer” yoki “.msi” faylini tanlang.[](https://www.java.com/en/download/help/java_win64bit.html)
   - Eng so‘nggi LTS (uzoq muddatli qo‘llab-quvvatlanadigan) versiya (masalan, JDK 21) yoki yangi versiyani (masalan, JDK 23) tanlang.[](https://phoenixnap.com/kb/install-java-windows)[](https://www.geeksforgeeks.org/download-install-java-windows-linux-macos/)
3. Oracle JDK uchun “Accept License Agreement” tugmasini bosing (agar talab qilinsa).
4. Yuklab olish tugmasini bosing va faylni kompyuteringizda ma’lum joyga (masalan, “Yuklamalar” papkasiga) saqlang. Yuklangan fayl nomi masalan, `jdk-21_windows-x64_bin.exe` yoki `jdk-23_windows-x64_bin.exe` bo‘lishi mumkin.[](https://docs.oracle.com/en/java/javase/21/install/installation-jdk-microsoft-windows-platforms.html)[](https://docs.oracle.com/en/java/javase/22/install/installation-jdk-microsoft-windows-platforms.html)

**Eslatma**: Yuklangan faylning hajmi va cheksummasini Oracle yoki Adoptium saytidagi ma’lumotlar bilan solishtirib, to‘g‘ri yuklanganligini tekshiring.[](https://docs.oracle.com/en/java/javase/11/install/installation-jdk-microsoft-windows-platforms.html)

## 2-qadam: JDK ni O‘rnatish

JDK ni o‘rnatish jarayoni oddiy va grafik interfeys orqali amalga oshiriladi. Quyidagi qadamlarni bajaring:

1. Yuklangan `.exe` yoki `.msi` faylini ikki marta bosing. Masalan, `jdk-21_windows-x64_bin.exe`.[](https://docs.oracle.com/en/java/javase/21/install/installation-jdk-microsoft-windows-platforms.html)
2. **User Account Control (UAC)** so‘rovi paydo bo‘lsa, “Yes” tugmasini bosing (administrator huquqlari talab qilinadi).[](https://www.whatismybrowser.com/guides/how-to-install-java/windows)
3. O‘rnatish ustasi (Installation Wizard) ochiladi. “Next” tugmasini bosing.
4. O‘rnatish joyini tanlang:
   - Standart yo‘l: `C:\Program Files\Java\jdk-<versiya>` (masalan, `C:\Program Files\Java\jdk-21`).
   - Tavsiya: Standart yo‘lni o‘zgartirmaslik yaxshiroq, ammo agar “Program Files” papkasidagi bo‘shliq skriptlarda muammo keltirib chiqarsa, masalan, `C:\Java\jdk-21` kabi oddiy yo‘lni tanlang.[](https://www.theserverside.com/blog/Coffee-Talk-Java-News-Stories-and-Opinions/How-do-I-install-Java-on-Windows)
5. O‘rnatish jarayoni tugagach, “Successfully Installed” xabari paydo bo‘ladi. “Close” tugmasini bosing.[](https://phoenixnap.com/kb/install-java-windows)[](https://medium.com/%40gauravshah97/how-to-install-java-on-windows-20fb8c329b74)

**Eslatma**: Agar internet firewall faol bo‘lsa, o‘rnatish jarayonida muammolar yuzaga kelishi mumkin. Zarur bo‘lsa, firewallni vaqtincha o‘chirib qo‘ying.[](https://www.java.com/en/download/help/windows_manual_download.html)

## 3-qadam: Muhit O‘zgaruvchilarini Sozlash

Java dasturlarini istalgan joydan kompilyatsiya qilish va ishga tushirish uchun `JAVA_HOME` va `Path` muhit o‘zgaruvchilarini sozlash kerak. Bu jarayon Java vositalarini (masalan, `java`, `javac`) tizim bo‘ylab ishlatish imkonini beradi.[](https://medium.com/%40gauravshah97/how-to-install-java-on-windows-20fb8c329b74)

### `JAVA_HOME` ni sozlash:
1. **Tizim sozlamalarini oching**:
   - Windows tugmasi + R ni bosing va `sysdm.cpl` yozing, so‘ng Enter’ni bosing. Yoki “This PC” ni o‘ng tugmasi bilan bosing va “Properties” > “Advanced system settings” ni tanlang.
2. “Environment Variables” tugmasini bosing.
3. “System Variables” bo‘limida “New” tugmasini bosing.
4. Quyidagi ma’lumotlarni kiriting:
   - **Variable name**: `JAVA_HOME`
   - **Variable value**: JDK o‘rnatilgan papka yo‘li, masalan, `C:\Program Files\Java\jdk-21`.[](https://medium.com/%40gauravshah97/how-to-install-java-on-windows-20fb8c329b74)[](https://devwithus.com/install-java-windows-10/)
5. “OK” ni bosing.

### `Path` ni yangilash:
1. “System Variables” bo‘limida “Path” ni toping va “Edit” tugmasini bosing.
2. “New” tugmasini bosing va `%JAVA_HOME%\bin` yo‘lini qo‘shing.
3. Barcha oynalarda “OK” ni bosib, sozlamalarni saqlang.

**Eslatma**: Agar eski Java versiyalari uchun `Path` da yo‘llar bo‘lsa, ularni o‘chirib, yangi versiyani qo‘shing.[](https://www.tutorials24x7.com/java/how-to-install-java-17-on-windows)

## 4-qadam: O‘rnatishni Tekshirish

JDK to‘g‘ri o‘rnatilganligini tekshirish uchun:
1. **Command Prompt** ni oching (Windows + R, `cmd` yozing va Enter’ni bosing).
2. Quyidagi buyruqni kiriting:
   ```bash
   java -version
   ```
3. Agar o‘rnatish muvaffaqiyatli bo‘lsa, quyidagiga o‘xshash natija chiqadi:
   ```
   java version "21.0.1" 2023-10-17 LTS
   Java(TM) SE Runtime Environment (build 21.0.1+12-LTS-29)
   Java HotSpot(TM) 64-Bit Server VM (build 21.0.1+12-LTS-29, mixed mode, sharing)
   ```
4. JDK vositalarini tekshirish uchun:
   ```bash
   javac -version
   ```
   Natija: `javac 21.0.1` kabi ko‘rinadi.

Agar “java is not recognized as an internal or external command” xabari chiqsa, `JAVA_HOME` yoki `Path` sozlamalarini qayta tekshiring.

## 5-qadam: Oddiy Java Dasturini Sinovdan O‘tkazish

O‘rnatishni sinash uchun oddiy Java dasturini yozib, kompilyatsiya qiling va ishga tushiring:

1. Matn muharririni (masalan, Notepad yoki Notepad++) oching.
2. Quyidagi kodni yozing va `HelloWorld.java` sifatida saqlang:
   ```java
   public class HelloWorld {
       public static void main(String[] args) {
           System.out.println("Salom, Dunyo!");
       }
   }
   ```
   **Eslatma**: Fayl nomini `HelloWorld.java` deb saqlash muhim, chunki sinf nomi bilan fayl nomi mos bo‘lishi kerak.
3. Command Prompt’da fayl saqlangan papkaga o‘ting:
   ```bash
   cd C:\Users\YourUsername\Desktop
   ```
4. Dasturni kompilyatsiya qiling:
   ```bash
   javac HelloWorld.java
   ```
   Bu `HelloWorld.class` faylini yaratadi.
5. Dasturni ishga tushiring:
   ```bash
   java HelloWorld
   ```
   Natija: `Salom, Dunyo!` so‘zi chiqadi.

## 6-qadam: Eski Versiyalarni O‘chirish (Ixtiyoriy)

Agar tizimda eski Java versiyalari bo‘lsa, ularni o‘chirish tavsiya qilinadi, chunki ular xavfsizlik muammolariga olib kelishi yoki yangi versiya bilan konfliktlar yaratishi mumkin.
1. **Control Panel** > “Programs and Features” ga o‘ting.
2. Eski JDK yoki JRE versiyalarini toping va “Uninstall” ni bosing.
3. Agar ro‘yxatdan keyin ham muammolar bo‘lsa, Microsoft’ning “Program Install and Uninstall Troubleshooter” vositasidan foydalaning.
