# Java’da Command-Line Arguments Haqida To‘liq Ma’lumot

**Command-line arguments** (buyruqlar qatori argumentlari) Java dasturini ishga tushirganda dasturga tashqi ma’lumotlarni uzatish imkonini beruvchi mexanizmdir. Ular dasturni ishga tushirishda buyruqlar qatori orqali kiritiladi va dastur ishlayotganda ulardan foydalanish mumkin. Quyida command-line argumentlarining maqsadi, foydalanishi va misollar batafsil tushuntiriladi.

---

## Command-Line Arguments Nima?

Command-line arguments Java dasturini `java` buyrug‘i bilan ishga tushirganda `main` metodiga uzatiladigan matnli (string) ma’lumotlardir. Ular `main` metodining `String[] args` parametri orqali qabul qilinadi. Masalan:

```java
public class Main {
    public static void main(String[] args) {
        // args bu yerda command-line argumentlarini saqlaydi
    }
}
```

Agar dastur `java Main arg1 arg2` buyrug‘i bilan ishga tushirilsa, `args` massivi `["arg1", "arg2"]` sifatida kiritiladi.

---

## Nima Uchun Command-Line Arguments Kerak?

Command-line argumentlari dasturga moslashuvchanlik va dinamiklik qo‘shadi. Quyida ularning asosiy foydalari keltirilgan:

1. **Dasturga Tashqi Ma’lumot Uzatish**:
   - Dastur ishlayotganda foydalanuvchi tomonidan kiritiladigan ma’lumotlarni qabul qilish imkonini beradi.
   - Masalan, fayl nomini, foydalanuvchi ismini yoki boshqa sozlamalarni uzatish mumkin.

2. **Moslashuvchanlik**:
   - Dastur har safar qayta kompilyatsiya qilinmasdan turli xil kiritishlar bilan ishlay oladi.
   - Masalan, dastur bir faylni o‘qishi yoki boshqa faylni o‘qishi uchun fayl nomini argument sifatida olishi mumkin.

3. **Avtomatlashtirish va Skriptlar**:
   - Skriptlar yoki avtomatlashtirilgan jarayonlarda ishlatilganda, argumentlar dasturga kerakli ma’lumotlarni avtomatik uzatish imkonini beradi.
   - Masalan, server sozlamalari yoki konfiguratsiya fayllari argument sifatida uzatilishi mumkin.

4. **Foydalanuvchi Interfeyssiz Ilovalar**:
   - Konsol ilovalari uchun foydalanuvchi bilan o‘zaro aloqa qilishning asosiy usuli sifatida ishlatiladi.
   - Masalan, faylni qayta ishlash dasturi foydalanuvchidan fayl yo‘lini argument sifatida oladi.

5. **Test va Debugging**:
   - Dastur sinovdan o‘tkazilayotganda turli sinov ma’lumotlarini uzatish uchun ishlatiladi.
   - Masalan, dastur turli parametrlar bilan sinovdan o‘tkazilishi mumkin.

## Command-Line Arguments Qanday Ishlatiladi?

Java’da command-line argumentlari `main` metodining `String[] args` parametri orqali qabul qilinadi. Quyida ulardan foydalanish jarayoni tushuntiriladi:

### Sintaksis:
```java
public class Main {
    public static void main(String[] args) {
        // args - String massivi, argumentlarni saqlaydi
        for (String arg : args) {
            System.out.println(arg);
        }
    }
}
```

- `args` — String turidagi massiv bo‘lib, unda buyruqlar qatorida kiritilgan barcha argumentlar saqlanadi.
- Argumentlar bo‘sh joy bilan ajratiladi. Masalan, `java Main arg1 arg2` buyrug‘i `args` massivida `["arg1", "arg2"]` sifatida saqlanadi.

### Dasturni ishga tushirish:
Dasturni ishga tushirish uchun:
1. Dasturni kompilyatsiya qiling:
   ```bash
   javac Main.java
   ```
2. Dasturni argumentlar bilan ishga tushiring:
   ```bash
   java Main arg1 arg2
   ```

### Misol 1: Argumentlarni Chop Etish
Quyidagi dastur barcha kiritilgan argumentlarni chop etadi:

```java
public class PrintArgs {
    public static void main(String[] args) {
        System.out.println("Kiritilgan argumentlar:");
        for (int i = 0; i < args.length; i++) {
            System.out.println("Argument " + (i + 1) + ": " + args[i]);
        }
    }
}
```

**Ishga tushirish**:
```bash
java PrintArgs Salom Dunyo 123
```

**Natija**:
```
Kiritilgan argumentlar:
Argument 1: Salom
Argument 2: Dunyo
Argument 3: 123
```

### Misol 2: Fayl Nomini Argument Sifatida Olish
Quyidagi dastur fayl nomini argument sifatida oladi va fayl mavjudligini tekshiradi:

```java
import java.io.File;

public class FileCheck {
    public static void main(String[] args) {
        if (args.length == 0) {
            System.out.println("Fayl nomi kiritilmadi!");
            return;
        }
        
        String fileName = args[0];
        File file = new File(fileName);
        
        if (file.exists()) {
            System.out.println("Fayl topildi: " + fileName);
        } else {
            System.out.println("Fayl topilmadi: " + fileName);
        }
    }
}
```

**Ishga tushirish**:
```bash
java FileCheck test.txt
```

**Natija** (agar `test.txt` mavjud bo‘lsa):
```
Fayl topildi: test.txt
```

**Natija** (agar fayl mavjud bo‘lmasa):
```
Fayl topilmadi: test.txt
```

## Command-Line Arguments bilan Ishlashda E’tibor Beriladigan Jihatlar

1. **Argumentlar Sonini Tekshirish**:
   - `args.length` yordamida kiritilgan argumentlar sonini tekshirish kerak, chunki foydalanuvchi hech qanday argument kiritmasligi mumkin.
   - Masalan:
     ```java
     if (args.length == 0) {
         System.out.println("Hech qanday argument kiritilmadi!");
         return;
     }
     ```

2. **Argument Turlari**:
   - Argumentlar har doim `String` sifatida kiritiladi. Agar raqam yoki boshqa turdagi ma’lumot kerak bo‘lsa, uni konvertatsiya qilish kerak:
     ```java
     int number = Integer.parseInt(args[0]); // String’ni int’ga aylantirish
     ```

3. **Xatolarni Boshqarish**:
   - Noto‘g‘ri formatdagi argumentlar uchun xatolarni boshqarish muhim:
     ```java
     try {
         int number = Integer.parseInt(args[0]);
         System.out.println("Raqam: " + number);
     } catch (NumberFormatException e) {
         System.out.println("Xato: Raqam kiritilmadi!");
     }
     ```

4. **Bo‘shliqlar va Maxsus Belgilar**:
   - Agar argumentda bo‘shliq bo‘lsa, uni qo‘shtirnoq ichiga olish kerak:
     ```bash
     java Main "Salom Dunyo"
     ```
   - Aks holda, bo‘shliqlar argumentlarni ajratadi.

5. **IDE’da Argumentlarni Sozlash**:
   - IntelliJ IDEA, Eclipse yoki VS Code kabi IDE’larda dasturni sinovdan o‘tkazish uchun argumentlarni sozlash mumkin:
     - IntelliJ IDEA: Run > Edit Configurations > Program arguments.
     - Eclipse: Run > Run Configurations > Arguments.
     - VS Code: `.vscode/launch.json` faylida `args` sozlamasini qo‘shing.
