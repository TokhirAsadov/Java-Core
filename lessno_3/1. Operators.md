# Java'da Operatorlar

Java'da **operatorlar** - bu dasturda ma'lumotlarni qayta ishlash, solishtirish yoki boshqa amallarni bajarish uchun ishlatiladigan maxsus belgilar yoki kalit so'zlar. Operatorlar odatda o'zgaruvchilar yoki qiymatlar ustida arifmetik, mantiqiy, solishtirish yoki boshqa operatsiyalarni bajaradi. Java'da operatorlar bir nechta toifalarga bo'linadi:

## 1. Arifmetik Operatorlar
Matnli yoki raqamli qiymatlar bilan matematik amallarni bajarish uchun ishlatiladi.
- `+` : Qo'shish (masalan, `5 + 3 = 8`)
- `-` : Ayirish (masalan, `5 - 3 = 2`)
- `*` : Ko'paytirish (masalan, `5 * 3 = 15`)
- `/` : Bo'lish (masalan, `6 / 2 = 3`)
- `%` : Modul (qoldiq, masalan, `7 % 2 = 1`)
- `++` : Increment (birlik qo'shish, masalan, `x++` yoki `++x`)
- `--` : Decrement (birlik ayirish, masalan, `x--` yoki `--x`)

**Misol:**
```java
int a = 10, b = 3;
System.out.println(a + b); // 13
System.out.println(a % b); // 1
System.out.println(a++);   // 10 (keyin 11 bo'ladi)
```

## 2. Solishtirish Operatorlari
Ikki qiymatni solishtirish uchun ishlatiladi, natija `true` yoki `false` bo'ladi.
- `==` : Tenglik (masalan, `a == b`)
- `!=` : Teng emas (masalan, `a != b`)
- `>` : Katta (masalan, `a > b`)
- `<` : Kichik (masalan, `a < b`)
- `>=` : Katta yoki teng (masalan, `a >= b`)
- `<=` : Kichik yoki teng (masalan, `a <= b`)

**Misol:**
```java
int x = 5, y = 3;
System.out.println(x > y);  // true
System.out.println(x == y); // false
```

## 3. Mantiqiy Operatorlar
Mantiqiy ifodalar bilan ishlash uchun ishlatiladi, odatda `boolean` qiymatlarga qo'llaniladi.
- `&&` : VA (AND, ikkala shart ham to'g'ri bo'lsa `true`)
- `||` : YOKI (OR, kamida bitta shart to'g'ri bo'lsa `true`)
- `!` : INKOR (NOT, mantiqiy qiymatni teskarisiga aylantiradi)
- `^` : XOR (eksklyuziv YOKI, faqat bitta shart to'g'ri bo'lsa `true`)

**Misol:**
```java
boolean a = true, b = false;
System.out.println(a && b); // false
System.out.println(a || b); // true
System.out.println(!a);     // false
```

## 4. Bitli Operatorlar
Bit darajasida amallar bajaradi (raqamlarning ikkilik shakli bilan ishlaydi).
- `&` : Bitli VA
- `|` : Bitli YOKI
- `^` : Bitli XOR
- `~` : Bitli INKOR
- `<<` : Chapga siljish
- `>>` : O'ngga siljish
- `>>>` : Imzosiz o'ngga siljish

**Misol:**
```java
int a = 5; // 0101 (ikkilikda)
int b = 3; // 0011 (ikkilikda)
System.out.println(a & b); // 1 (0101 & 0011 = 0001)
System.out.println(a << 1); // 10 (0101 << 1 = 1010)
```

## 5. Tayinlash Operatorlari
O'zgaruvchilarga qiymat tayinlash uchun ishlatiladi.
- `=` : Oddiy tayinlash (masalan, `x = 5`)
- `+=` : Qo'shib tayinlash (masalan, `x += 3` → `x = x + 3`)
- `-=` : Ayirib tayinlash
- `*=` : Ko'paytirib tayinlash
- `/=` : Bo'lib tayinlash
- `%=` : Modul qilib tayinlash
- Bitli tayinlashlar: `&=`, `|=`, `^=`, `<<=`, `>>=`, `>>>=`

**Misol:**
```java
int x = 10;
x += 5; // x = x + 5
System.out.println(x); // 15
```

## 6. Shartli (Ternary) Operator
Shart asosida qiymat qaytaradi: `shart ? qiymat1 : qiymat2`
- Agar `shart` to'g'ri bo'lsa, `qiymat1` qaytariladi, aks holda `qiymat2`.

**Misol:**
```java
int a = 10, b = 20;
int max = (a > b) ? a : b;
System.out.println(max); // 20
```

## 7. Boshqa Operatorlar
- **`instanceof`**: Ob'ektning ma'lum bir sinfga tegishli ekanligini tekshiradi.
  ```java
  String s = "Salom";
  System.out.println(s instanceof String); // true
  ```
- **`.` (nuqta)**: Ob'ektning metodlari yoki xususiyatlariga kirish uchun.
  ```java
  System.out.println("Salom".length()); // 5
  ```
- **`[]`**: Massiv elementlariga kirish uchun.
  ```java
  int[] arr = {1, 2, 3};
  System.out.println(arr[0]); // 1
  ```

## 8. Operatorlarning Ustuvorligi
Operatorlarning qaysi biri birinchi bajarilishi ularning ustuvorligiga bog'liq. Masalan:
- `()` (qavslar) eng yuqori ustuvorlikka ega.
- `*`, `/`, `%` ko'paytirish va bo'lish operatorlari `+`, `-` dan oldin bajariladi.
- Mantiqiy `&&` operatorlari `||` dan oldin bajariladi.

**Misol:**
```java
int x = 2 + 3 * 4; // 3 * 4 = 12, keyin 2 + 12 = 14
System.out.println(x); // 14
```

## Xulosa
Java'da operatorlar dasturlarni dinamik va samarali qilish uchun muhim vositadir. Ular arifmetik, mantiqiy, bitli va boshqa ko'plab operatsiyalarni bajaradi. Har bir operatorning o'ziga xos maqsadi bor va to'g'ri ishlatilishi dastur sifatini oshiradi.
