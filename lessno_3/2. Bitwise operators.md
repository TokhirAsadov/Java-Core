# Java'da Shifting Operatorlari

Java'da **shifting operatorlari** (siljish operatorlari) raqamlarning ikkilik (binary) shaklidagi bitlarini chapga yoki o'ngga siljitish uchun ishlatiladi. Bu operatorlar asosan `int` va `long` kabi butun son turlari bilan ishlaydi va bit darajasida manipulyatsiya qilishda foydalaniladi. Java'da uchta shifting operatori mavjud: **chapga siljish (`<<`)**, **o'ngga siljish (`>>`)** va **imzosiz o'ngga siljish (`>>>`)**.

## 1. Chapga Siljish Operatori (`<<`)
- **Ta'rifi**: Raqamning bitlarini chapga siljitadi, o'ngdan bo'shagan joylarga nollar qo'shiladi.
- **Effekti**: Har bir chapga siljish raqamni 2 ga ko'paytirishga teng (agar to'lib ketish bo'lmasa).
- **Sintaksis**: `qiymat << siljish_soni`

**Misol:**
```java
int a = 5; // Ikkilik: 00000101
int result = a << 2; // 2 ta bit chapga siljitish
System.out.println(result); // Natija: 20 (Ikkilik: 00010100)
```
- **Tushuntirish**: `5 << 2` → `00000101` chapga 2 bit siljiydi → `00010100` (10lik shaklda 20).

## 2. O'ngga Siljish Operatori (`>>`)
- **Ta'rifi**: Raqamning bitlarini o'ngga siljitadi, chapdan bo'shagan joylarga raqamning **ishora biti** (sign bit) qo'shiladi (ya'ni manfiy sonlar uchun 1, musbat sonlar uchun 0).
- **Effekti**: Har bir o'ngga siljish raqamni 2 ga bo'lishga teng (manfiy sonlarda ishora saqlanadi).
- **Sintaksis**: `qiymat >> siljish_soni`

**Misol:**
```java
int a = 20; // Ikkilik: 00010100
int result = a >> 2; // 2 ta bit o'ngga siljitish
System.out.println(result); // Natija: 5 (Ikkilik: 00000101)
```
- **Manfiy son bilan misol:**
```java
int a = -20; // Ikkilik: 11101100 (32-bit)
int result = a >> 2; // 2 ta bit o'ngga siljitish
System.out.println(result); // Natija: -5 (Ikkilik: 11111101)
```
- **Tushuntirish**: Manfiy sonlarda chapdan 1 qo'shiladi, bu ishorani saqlaydi.

## 3. Imzosiz O'ngga Siljish Operatori (`>>>`)
- **Ta'rifi**: Raqamning bitlarini o'ngga siljitadi, lekin chapdan bo'shagan joylarga har doim **nollar** qo'shiladi, ya'ni ishora bitiga e'tibor berilmaydi.
- **Effekti**: Manfiy sonlarda ham musbat natija chiqishi mumkin, chunki ishora biti yo'qotiladi.
- **Sintaksis**: `qiymat >>> siljish_soni`

**Misol:**
```java
int a = -20; // Ikkilik: 11101100 (32-bit)
int result = a >>> 2; // 2 ta bit o'ngga siljitish
System.out.println(result); // Natija: 1073741819 (Ikkilik: 00111011...)
```
- **Tushuntirish**: `>>>` manfiy sonni musbat sonda aylantirishi mumkin, chunki chapdan doimo 0 qo'shiladi.

## 4. Shifting Operatorlarining Xususiyatlari
- **Tezlik**: Shifting operatorlari ko'paytirish (`*`) yoki bo'lish (`/`) operatorlariga nisbatan ancha tezroq ishlaydi, chunki ular to'g'ridan-to'g'ri bitlar bilan ishlaydi.
- **Chegaralar**: 
  - Agar siljish soni ma'lumot turining bit uzunligidan (masalan, `int` uchun 32 bit) katta bo'lsa, faqat `siljish_soni % 32` hisobga olinadi.
  - Masalan: `a << 34` → `a << 2` deb hisoblanadi (`34 % 32 = 2`).
- **Ma'lumot yo'qotilishi**: O'ngga siljitishda kichik bitlar yo'qoladi, chapga siljitishda esa katta bitlar yo'qolishi mumkin.

## 5. Foydalanish Holatlari
- **Optimallashtirish**: Ko'paytirish yoki bo'lish o'rniga shifting operatorlaridan foydalanish (masalan, `x * 4` o'rniga `x << 2`).
- **Bit manipulyatsiyasi**: Ma'lumotlarni siqish, shifrlash yoki maxsus algoritmlarda bitlar bilan ishlash.
- **Binar operatsiyalar**: Ma'lum bitlarni o'rnatish yoki o'chirish (bit maskalari bilan birgalikda).

**Misol (Bit manipulyatsiyasi):**
```java
int a = 5; // Ikkilik: 00000101
int b = a << 1; // Ikkilik: 00001010 (10)
int c = a >> 1; // Ikkilik: 00000010 (2)
int d = -20 >>> 1; // Ikkilik: 0111... (musbat son)
System.out.println("Chapga siljish: " + b); // 10
System.out.println("O'ngga siljish: " + c); // 2
System.out.println("Imzosiz o'ngga siljish: " + d); // 2147483639
```

## 6. E'tibor Beriladigan Jihatlar
- **Manfiy sonlar bilan ehtiyotkorlik**: `>>` va `>>>` operatorlari manfiy sonlarda farqli natija beradi.
- **To'lib ketish (overflow)**: Chapga siljitishda katta qiymatlar chegaradan oshib ketishi mumkin.
- **char bilan ishlash**: `char` turi ham `int` ga avtomatik cast qilinadi va shifting operatorlari bilan ishlatilishi mumkin.
  ```java
  char c = 'A'; // Unicode: 65 (00000000 01000001)
  int result = c << 1;
  System.out.println(result); // 130 (00000000 10000010)
  ```

## Xulosa
Java'da shifting operatorlari (`<<`, `>>`, `>>>`) bitlar bilan samarali ishlash uchun ishlatiladi. Ular ko'paytirish va bo'lishni optimallashtirish, bit manipulyatsiyasi va maxsus algoritmlarda qo'llaniladi. `<<` raqamni ko'paytirishga, `>>` ishorani saqlagan holda bo'lishga, `>>>` esa ishorasiz bo'lishga xizmat qiladi. To'g'ri ishlatilganda ular dastur samaradorligini oshiradi.
