# Java'da Matematik Funksiyalar va Konstantalar

Java'da **matematik funksiyalar va konstantalar** asosan `java.lang.Math` classi orqali taqdim etiladi. Bu class matematik operatsiyalarni bajarish uchun ko'plab statik metodlar va konstantalarni o'z ichiga oladi. `Math` classi `java.lang` paketida joylashgan bo'lib, uni import qilish shart emas. Quyida matematik funksiyalar va konstantalar haqida to'liq ma'lumot beriladi.

## 1. Matematik Konstantalar
`Math` classi ikkita asosiy matematik konstantani taqdim etadi:
- **`Math.PI`**: Π (pi) konstantasi, taxminan 3.141592653589793.
  - **Foydalanish**: Daira uzunligi, maydoni yoki boshqa geometrik hisob-kitoblar uchun.
- **`Math.E`**: Exponensial funksiyaning asosi (e), taxminan 2.718281828459045.
  - **Foydalanish**: Logarifmik va eksponensial hisob-kitoblar uchun.

**Misol:**
```java
System.out.println("PI: " + Math.PI); // Natija: 3.141592653589793
System.out.println("E: " + Math.E);   // Natija: 2.718281828459045
```

## 2. Matematik Funksiyalar
`Math` classidagi funksiyalar statik bo'lib, ularni class nomi orqali to'g'ridan-to'g'ri chaqirish mumkin (masalan, `Math.abs()`). Quyida asosiy matematik funksiyalar guruhlarga bo'linib tushuntiriladi:

### 2.1. Asosiy Arifmetik Funksiyalar
- **`Math.abs(x)`**: `x` ning absolyut qiymatini qaytaradi (manfiy bo'lsa musbat qiladi).
  - **Turlari**: `int`, `long`, `float`, `double`.
  - **Misol**: `Math.abs(-5.7)` → `5.7`
- **`Math.max(a, b)`**: `a` va `b` dan kattasini qaytaradi.
  - **Turlari**: `int`, `long`, `float`, `double`.
  - **Misol**: `Math.max(10, 20)` → `20`
- **`Math.min(a, b)`**: `a` va `b` dan kichigini qaytaradi.
  - **Turlari**: `int`, `long`, `float`, `double`.
  - **Misol**: `Math.min(10, 20)` → `10`

**Misol:**
```java
System.out.println(Math.abs(-10));    // Natija: 10
System.out.println(Math.max(5, 12));  // Natija: 12
System.out.println(Math.min(5, 12));  // Natija: 5
```

### 2.2. Yaxlitlash Funksiyalari
- **`Math.round(x)`**: `x` ni eng yaqin butun songa yaxlitlaydi.
  - **Qaytaradi**: `int` yoki `long` (agar `x` `float` yoki `double` bo'lsa).
  - **Misol**: `Math.round(5.6)` → `6`, `Math.round(5.4)` → `5`
- **`Math.floor(x)`**: `x` ni eng yaqin kichik butun songa yaxlitlaydi (pastga).
  - **Qaytaradi**: `double`.
  - **Misol**: `Math.floor(5.9)` → `5.0`
- **`Math.ceil(x)`**: `x` ni eng yaqin katta butun songa yaxlitlaydi (yuqoriga).
  - **Qaytaradi**: `double`.
  - **Misol**: `Math.ceil(5.1)` → `6.0`
- **`Math.rint(x)`**: `x` ni eng yaqin butun songa yaxlitlaydi, agar teng masofa bo'lsa juft sonni tanlaydi.
  - **Qaytaradi**: `double`.
  - **Misol**: `Math.rint(5.5)` → `6.0`, `Math.rint(4.5)` → `4.0`

**Misol:**
```java
System.out.println(Math.round(5.6)); // Natija: 6
System.out.println(Math.floor(5.9)); // Natija: 5.0
System.out.println(Math.ceil(5.1));  // Natija: 6.0
System.out.println(Math.rint(5.5));  // Natija: 6.0
```

### 2.3. Eksponensial va Logarifmik Funksiyalar
- **`Math.exp(x)`**: `e^x` ni hisoblaydi.
  - **Misol**: `Math.exp(1)` → `2.718281828459045` (taxminan `Math.E`)
- **`Math.log(x)`**: `x` ning natural logarifmini (e asosida) hisoblaydi.
  - **Misol**: `Math.log(Math.E)` → `1.0`
- **`Math.log10(x)`**: `x` ning 10 asosidagi logarifmini hisoblaydi.
  - **Misol**: `Math.log10(100)` → `2.0`
- **`Math.pow(a, b)`**: `a` ning `b` darajasini hisoblaydi (`a^b`).
  - **Misol**: `Math.pow(2, 3)` → `8.0`
- **`Math.sqrt(x)`**: `x` ning kvadrat ildizini hisoblaydi.
  - **Misol**: `Math.sqrt(16)` → `4.0`
- **`Math.cbrt(x)`**: `x` ning kub ildizini hisoblaydi.
  - **Misol**: `Math.cbrt(8)` → `2.0`

**Misol:**
```java
System.out.println(Math.exp(1));     // Natija: 2.718281828459045
System.out.println(Math.log(Math.E)); // Natija: 1.0
System.out.println(Math.pow(2, 3));  // Natija: 8.0
System.out.println(Math.sqrt(16));   // Natija: 4.0
```

### 2.4. Trigonometrik Funksiyalar
Barcha trigonometrik funksiyalar radianlarda ishlaydi (daraja emas).
- **`Math.sin(x)`**: `x` radianning sinusi.
- **`Math.cos(x)`**: `x` radianning kosinusi.
- **`Math.tan(x)`**: `x` radianning tangensi.
- **`Math.asin(x)`**: `x` ning arksinusi (sinusning teskarisi).
- **`Math.acos(x)`**: `x` ning arkkosinusi.
- **`Math.atan(x)`**: `x` ning arktangensi.
- **`Math.atan2(y, x)`**: `(x, y)` koordinatalarining burchagini hisoblaydi.

**Misol:**
```java
double angle = Math.toRadians(90); // 90 darajani radianlarga aylantirish
System.out.println(Math.sin(angle)); // Natija: 1.0
System.out.println(Math.cos(angle)); // Natija: 0.0 (taxminan)
```

### 2.5. Boshqa Foydali Funksiyalar
- **`Math.toRadians(degrees)`**: Darajalarni radianlarga aylantiradi.
  - **Misol**: `Math.toRadians(180)` → `3.141592653589793`
- **`Math.toDegrees(radians)`**: Radianlarni darajalarga aylantiradi.
  - **Misol**: `Math.toDegrees(Math.PI)` → `180.0`
- **`Math.random()`**: 0.0 (shu jumladan) va 1.0 (undan tashqari) oralig'ida tasodifiy `double` qiymat qaytaradi.
  - **Misol**: `Math.random()` → `0.0` dan `1.0` gacha tasodifiy son
- **`Math.hypot(x, y)`**: Gipotenuzani hisoblaydi (`sqrt(x^2 + y^2)`).
  - **Misol**: `Math.hypot(3, 4)` → `5.0`

**Misol:**
```java
System.out.println(Math.toDegrees(Math.PI)); // Natija: 180.0
System.out.println(Math.random());           // Natija: 0.0..1.0 oralig'ida tasodifiy son
System.out.println(Math.hypot(3, 4));       // Natija: 5.0
```

## 3. E'tibor Beriladigan Jihatlar
- **Statik metodlar**: `Math` classidagi barcha funksiyalar statik, ya'ni ob'ekt yaratmasdan `Math.<metod>` shaklida ishlatiladi.
- **Radianlar**: Trigonometrik funksiyalar radianlarda ishlaydi. Darajalarni radianlarga aylantirish uchun `Math.toRadians()` ishlatiladi.
- **Ma'lumot turi**: Ko'p funksiyalar `double` qiymat qaytaradi. Agar boshqa tur kerak bo'lsa, cast qilish talab qilinadi.
  ```java
  double d = Math.sqrt(16); // 4.0
  int i = (int) d;          // 4
  ```
- **Xatolar**: Ba'zi funksiyalar (masalan, `Math.log(-1)` yoki `Math.sqrt(-1)`) `NaN` (Not a Number) yoki `Infinity` qaytaradi, agar kirish qiymati noto'g'ri bo'lsa.

## 4. Amaliy Misol (To'liq dastur)
```java
public class MathExample {
    public static void main(String[] args) {
        // Konstantalar
        System.out.println("PI: " + Math.PI); // 3.141592653589793
        System.out.println("E: " + Math.E);   // 2.718281828459045

        // Arifmetik funksiyalar
        System.out.println("Absolyut: " + Math.abs(-10.5)); // 10.5
        System.out.println("Maksimum: " + Math.max(5, 12)); // 12
        System.out.println("Minimum: " + Math.min(5, 12)); // 5

        // Yaxlitlash
        System.out.println("Round: " + Math.round(5.6)); // 6
        System.out.println("Floor: " + Math.floor(5.9)); // 5.0
        System.out.println("Ceil: " + Math.ceil(5.1));  // 6.0

        // Eksponensial va logarifmik
        System.out.println("e^1: " + Math.exp(1));      // 2.718281828459045
        System.out.println("log(e): " + Math.log(Math.E)); // 1.0
        System.out.println("2^3: " + Math.pow(2, 3));   // 8.0
        System.out.println("sqrt(16): " + Math.sqrt(16)); // 4.0

        // Trigonometrik
        double angle = Math.toRadians(90);
        System.out.println("sin(90°): " + Math.sin(angle)); // 1.0
        System.out.println("cos(90°): " + Math.cos(angle)); // 0.0 (taxminan)

        // Tasodifiy son
        System.out.println("Tasodifiy: " + Math.random()); // 0.0..1.0 oralig'ida
    }
}
```

## 5. Foydalanish Holatlari
- **Matematik hisob-kitoblar**: Daira maydoni (`Math.PI * r * r`), kvadrat ildiz, darajaga ko'tarish.
- **Fizika va muhandislik**: Vektorlar, burchaklar, trayektoriyalar hisoblash.
- **Tasodifiy sonlar**: O'yinlar, simulyatsiyalar yoki testlar uchun `Math.random()`.
- **Optimallashtirish**: `Math.min()` va `Math.max()` orqali eng katta/kichik qiymatlarni aniqlash.

## Xulosa
Java'da `Math` classi matematik hisob-kitoblar uchun keng ko'lamli funksiyalar va konstantalarni taqdim etadi. `Math.PI` va `Math.E` konstantalari, shuningdek, arifmetik, yaxlitlash, eksponensial, logarifmik va trigonometrik funksiyalar dasturchilarga turli sohalarda qulayliklar yaratadi. Statik metodlar sifatida ularni ishlatish oson va samarali bo'lib, qavslar va to'g'ri kirish qiymatlari bilan xatosiz natijalar olish mumkin.
