# Java Tokens

## Java Tokenlari nima?
Java dasturlash tilida **tokenlar** – bu Java kompilyatori tomonidan aniqlanadigan va dasturda ma’noga ega bo‘lgan eng kichik individual elementlardir. Tokenlar dastur kodini tashkil qiluvchi asosiy qismlar bo‘lib, kompilyator ularni tahlil qilib, dastur sintaksisini tushunadi. Har bir token dasturning ma’lum bir qismini ifodalaydi, masalan, kalit so‘z, o‘zgaruvchi nomi, operator yoki ajratuvchi.

## Java Tokenlarining turlari
Java’da tokenlar quyidagi asosiy toifalarga bo‘linadi:

### 1. Kalit so‘zlar (Keywords)
- **Ta’rif**: Kalit so‘zlar – Java tilida maxsus ma’noga ega bo‘lgan, oldindan belgilangan so‘zlar. Ular dastur sintaksisida muayyan vazifalarni bajarish uchun ishlatiladi va o‘zgaruvchi yoki metod nomlari sifatida ishlatilmaydi.
- **Misollar**: `class`, `public`, `static`, `void`, `if`, `else`, `for`, `while`, `return`, `int`, `double`, `try`, `catch`.
- **Xususiyatlari**:
  - Java’da 50 ga yaqin kalit so‘z mavjud (masalan, `true`, `false`, `null` ham kalit so‘z hisoblanadi).
  - Kalit so‘zlar katta-kichik harflarga sezgir (case-sensitive) va faqat kichik harflarda yoziladi.
- **Misol**:
  ```java
  public class MyClass {
      // public va class – kalit so‘zlar
  }
  ```

### 2. Identifikatorlar (Identifiers)
- **Ta’rif**: Identifikatorlar – dasturchi tomonidan o‘zgaruvchilar, metodlar, sinflar yoki boshqa ob’ektlarga qo‘yilgan nomlardir.
- **Qoidalar**:
  - Harf (A-Z, a-z), pastki chiziq (`_`) yoki dollar belgisi (`$`) bilan boshlanishi kerak.
  - Raqamlar (0-9) ishlatilishi mumkin, lekin identifikator raqam bilan boshlanmaydi.
  - Kalit so‘zlar identifikator sifatida ishlatilmaydi.
  - Katta-kichik harflarga sezgir (masalan, `myVar` va `myvar` – ikki xil identifikator).
- **Misollar**: `myVariable`, `calculateSum`, `MyClass`, `$_value`.
- **Misol**:
  ```java
  int studentAge = 20; // studentAge – identifikator
  ```

### 3. Literallar (Literals)
- **Ta’rif**: Literallar – dasturda o‘zgarmas qiymatlarni ifodalovchi tokenlardir. Ular o‘zgaruvchilarga yoki operatsiyalarga qiymat sifatida beriladi.
- **Turlari**:
  - **Butun sonli literallar**: `123`, `-456` (10-lik), `0x1A` (16-lik), `0b1010` (2-lik).
  - **O‘nlik sonli literallar**: `3.14`, `0.001`, `2.5e3` (eksponensial shaklda).
  - **Belgili literallar**: `'a'`, `'\n'` (yangi qator), `'\u0041'` (Unicode belgisi).
  - **Satrli literallar**: `"Hello, World!"`, `""` (bo‘sh satr).
  - **Mantiqiy literallar**: `true`, `false`.
  - **Null literali**: `null` (ob’ekt yo‘qligini bildiradi).
- **Misol**:
  ```java
  int number = 100; // 100 – butun sonli literal
  String greeting = "Salom"; // "Salom" – satrli literal
  ```

### 4. Operatorlar (Operators)
- **Ta’rif**: Operatorlar – ma’lumotlarni qayta ishlash yoki operatsiyalarni bajarish uchun ishlatiladigan maxsus belgilar.
- **Turlari**:
  - **Aritmetik operatorlar**: `+`, `-`, `*`, `/`, `%` (modul).
  - **Taqqoslash operatorlari**: `==`, `!=`, `>`, `<`, `>=`, `<=`.
  - **Mantiqiy operatorlar**: `&&`, `||`, `!`.
  - **Bitli operatorlar**: `&`, `|`, `^`, `~`, `<<`, `>>`, `>>>`.
  - **Tayinlash operatorlari**: `=`, `+=`, `-=`, `*=`, `/=`.
  - **Boshqa operatorlar**: `instanceof`, `new`, `->` (lambda ifodasi).
- **Misol**:
  ```java
  int sum = a + b; // = va + – operatorlar
  ```

### 5. Ajratuvchilar (Separators)
- **Ta’rif**: Ajratuvchilar – kodni tuzilishini aniqlash uchun ishlatiladigan maxsus belgilar.
- **Misollar**:
  - Qavslar: `()` – metod chaqiruvlari va ifodalar uchun.
  - Jingalak qavslar: `{}` – kod bloklari uchun.
  - Kvadrat qavslar: `[]` – massivlar uchun.
  - Vergul: `,` – ro‘yxatlarni ajratish uchun.
  - Nuqta: `.` – ob’ektlar yoki sinf a’zolariga kirish uchun.
  - Nuqta-vergul: `;` – operatorlarni yakunlash uchun.
- **Misol**:
  ```java
  String[] names = {"Ali", "Vali"}; // [], {}, , – ajratuvchilar
  ```

### 6. Izohlar (Comments)
- **Ta’rif**: Izohlar – dastur kodini tushuntirish yoki o‘qishni osonlashtirish uchun ishlatiladi. Ular kompilyator tomonidan e’tiborga olinmaydi, lekin token sifatida qaraladi.
- **Turlari**:
  - **Bir qatorli izoh**: `// Bu bir qatorli izoh`
  - **Ko‘p qatorli izoh**: `/* Bu ko‘p qatorli izoh */`
  - **Javadoc izohi**: `/** Bu Javadoc izohi */` – API hujjatlarini yaratish uchun.
- **Misol**:
  ```java
  // Bu bir qatorli izoh
  /* Bu ko‘p qatorli
     izoh */
  /** Bu Javadoc izohi */
  ```
