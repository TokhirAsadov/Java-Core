# Java Comments

Java dasturlash tilida comments kodni tushuntirish, o‘qishni osonlashtirish yoki dastur qismlarini hujjatlashtirish uchun ishlatiladi. Izohlar Java kompilyatori tomonidan e’tiborga olinmaydi, ya’ni ular dastur ishlashiga ta’sir qilmaydi, lekin kodning ma’nosini tushunishga yordam beradi. Java’da uch xil izoh turi mavjud: bir qatorli izoh, ko‘p qatorli izoh va Javadoc izohi.

## 1. Bir qatorli izoh (Single Line Comment)
- **Ta’rif**: Bir qatorli izoh `//` belgisi bilan boshlanadi va qator oxirigacha davom etadi. U qisqa tushuntirishlar yoki eslatmalar uchun ishlatiladi.
- **Xususiyatlari**:
  - Faqat bitta qatorni qamrab oladi.
  - Kodning ma’lum bir qismini tushuntirish uchun qulay.
- **Misol**:
  ```java
  // Bu bir qatorli izoh
  int x = 5; // x o‘zgaruvchisiga 5 qiymati berilmoqda
  ```

## 2. Ko‘p qatorli izoh (Multi Line Comment)
- **Ta’rif**: Ko‘p qatorli izoh `/*` bilan boshlanadi va `*/` bilan yakunlanadi. U bir nechta qatorlarni qamrab olishi mumkin va uzunroq tushuntirishlar uchun ishlatiladi.
- **Xususiyatlari**:
  - Bir nechta qatorlarga izoh yozish imkonini beradi.
  - Kodning katta qismlarini vaqtincha “o‘chirib qo‘yish” (commenting out) uchun ham ishlatiladi.
- **Misol**:
  ```java
  /* Bu ko‘p qatorli izoh.
     U bir nechta qatorlarni qamrab oladi.
     Quyida oddiy o‘zgaruvchi e’lon qilinmoqda.
  */
  int y = 10;
  ```

## 3. Java dokumentatsiya izohi (Java Documentation Comment)
- **Ta’rif**: Javadoc izohlari `/**` bilan boshlanadi va `*/` bilan yakunlanadi. Ular dastur elementlari (sinflar, metodlar, o‘zgaruvchilar) haqida hujjatlashtirish uchun ishlatiladi. Javadoc vositasi yordamida bu izohlardan API hujjatlari yaratiladi.
- **Xususiyatlari**:
  - Odatda sinf, metod yoki o‘zgaruvchi haqida umumiy ma’lumot beradi.
  - Maxsus teglar (masalan, `@param`, `@return`, `@author`) ishlatiladi.
  - HTML hujjatlarini avtomatik yaratish uchun ishlatiladi.
- **Misol**:
  ```java
  /**
   * Bu sinf foydalanuvchi ma’lumotlarini saqlash uchun ishlatiladi.
   * @author Dasturchi Ismi
   * @version 1.0
   */
  public class User {
      /**
       * Foydalanuvchi ismini qaytaradi.
       * @param name Foydalanuvchi ismi
       * @return Foydalanuvchi ismini String sifatida qaytaradi
       */
      public String getName(String name) {
          return name;
      }
  }
  ```

## Qo‘shimcha maslahatlar
- **Bir qatorli izohlar**: Qisqa va aniq tushuntirishlar uchun ishlatiladi.
- **Ko‘p qatorli izohlar**: Kodning katta qismlarini tushuntirish yoki vaqtincha kodni “o‘chirish” uchun foydali.
- **Javadoc izohlari**: Professional dasturlarda API hujjatlarini yaratish uchun muhim. Har bir metod va sinf uchun aniq va to‘liq ma’lumot yozish tavsiya etiladi.
- **Ehtiyotkorlik**: Ortiqcha izohlardan saqlaning, chunki bu kodni o‘qishni qiyinlashtirishi mumkin.
