# Java’ning Primitiv Ma'lumot Turlari

## Java - strongly typed dasturlash tili
Java – **strongly typed** dasturlash tili hisoblanadi. Bu shuni anglatadiki, har bir o‘zgaruvchi e’lon qilinganda uning ma'lumot turi aniq ko‘rsatilishi kerak. O‘zgaruvchi bir marta ma'lumot turi bilan e’lon qilinganidan so‘ng, unga faqat o‘sha turdagi qiymatlar yuklanishi mumkin. Bu xususiyat xavfsizlikni oshiradi va xatolarni oldini olishga yordam beradi. Masalan:

```java
int number = 10; // number o‘zgaruvchisi int turida e’lon qilinadi
number = "Salom"; // Xato! number faqat butun sonlarni qabul qila oladi
```

Java’da sakkizta **primitiv ma'lumot turlari** mavjud bo‘lib, ular quyidagi toifalarga bo‘linadi:
1. To‘rtta butun son turlari: `byte`, `short`, `int`, `long`
2. Ikkita o‘nlik son turlari: `float`, `double`
3. Bitta belgi turi: `char`
4. Bitta mantiqiy tur: `boolean`

## Butun son turlari (Integer Types)
Java’da to‘rtta butun son ma'lumot turi mavjud. Ular o‘zgaruvchilarga butun sonlarni (hech qanday o‘nlik qismi bo‘lmagan sonlarni) saqlash uchun ishlatiladi. Har bir tur o‘zining xotira hajmi va qiymatlar diapazoniga ega.

### a) `byte`
- **Ta’rif**: Eng kichik butun son turi. 8 bit (1 bayt) xotira egallaydi.
- **Diapazon**: -128 dan +127 gacha.
- **Standart qiymat**: 0
- **Foydalanish**: Kichik sonlarni saqlashda, masalan, xotirani tejash zarur bo‘lganda ishlatiladi.
- **Misol**:
  ```java
  byte age = 25;
  byte temperature = -10;
  System.out.println(age); // Natija: 25
  ```

### b) `short`
- **Ta’rif**: 16 bit (2 bayt) xotira egallaydi.
- **Diapazon**: -32,768 dan +32,767 gacha.
- **Standart qiymat**: 0
- **Foydalanish**: O‘rtacha kattalikdagi sonlarni saqlashda ishlatiladi, lekin kamdan-kam qo‘llaniladi.
- **Misol**:
  ```java
  short population = 15000;
  System.out.println(population); // Natija: 15000
  ```

### c) `int`
- **Ta’rif**: Eng ko‘p ishlatiladigan butun son turi. 32 bit (4 bayt) xotira egallaydi.
- **Diapazon**: -2,147,483,648 dan +2,147,483,647 gacha.
- **Standart qiymat**: 0
- **Foydalanish**: Umumiy butun son operatsiyalari uchun standart tanlov.
- **Misol**:
  ```java
  int salary = 50000;
  System.out.println(salary); // Natija: 50000
  ```

### d) `long`
- **Ta’rif**: Eng katta butun son turi. 64 bit (8 bayt) xotira egallaydi.
- **Diapazon**: -9,223,372,036,854,775,808 dan +9,223,372,036,854,775,807 gacha.
- **Standart qiymat**: 0L
- **Foydalanish**: Katta sonlar, masalan, bank hisoblari yoki vaqt belgilari (timestamps) uchun ishlatiladi. Qiymatlar `L` yoki `l` suffiksi bilan yoziladi.
- **Misol**:
  ```java
  long distance = 1234567890123L;
  System.out.println(distance); // Natija: 1234567890123
  ```

## O‘nlik son turlari (Floating-Point Types)
Java’da ikkita o‘nlik son (haqiqiy sonlar) turi mavjud. Ular o‘nlik qismga ega sonlarni saqlash uchun ishlatiladi, masalan, 3.14 yoki -0.001.

### a) `float`
- **Ta’rif**: 32 bit (4 bayt) xotira egallaydi. Taxminan 6-7 o‘nlik raqamgacha aniqlik bilan ishlaydi.
- **Diapazon**: ±3.4 × 10³⁸ (taxminan).
- **Standart qiymat**: 0.0f
- **Foydalanish**: Kichikroq aniqlik talab qilinadigan o‘nlik sonlar uchun ishlatiladi. Qiymatlar `f` yoki `F` suffiksi bilan yoziladi.
- **Misol**:
  ```java
  float price = 19.99f;
  System.out.println(price); // Natija: 19.99
  ```

### b) `double`
- **Ta’rif**: 64 bit (8 bayt) xotira egallaydi. Taxminan 15 o‘nlik raqamgacha aniqlik bilan ishlaydi.
- **Diapazon**: ±1.7 × 10³⁰⁸ (taxminan).
- **Standart qiymat**: 0.0
- **Foydalanish**: Yuqori aniqlik talab qilinadigan o‘nlik sonlar uchun standart tanlov. Ilmiy hisob-kitoblar va katta o‘nlik sonlar uchun ishlatiladi.
- **Misol**:
  ```java
  double pi = 3.14159265359;
  System.out.println(pi); // Natija: 3.14159265359
  ```

## Belgi turi (Character Type)
### `char`
- **Ta’rif**: Bitta Unicode belgisini saqlash uchun ishlatiladi. 16 bit (2 bayt) xotira egallaydi.
- **Diapazon**: `\u0000` (0) dan `\uFFFF` (65,535) gacha, ya’ni Unicode belgi to‘plamidagi barcha belgilarni qamrab oladi.
- **Standart qiymat**: `\u0000` (null belgi).
- **Foydalanish**: Harflar, raqamlar, maxsus belgilar yoki Unicode belgilarni saqlash uchun ishlatiladi. Belgilar `''` ichida yoziladi.
- **Misol**:
  ```java
  char letter = 'A';
  char unicodeChar = '\u0041'; // Unicode’da A harfi
  System.out.println(letter); // Natija: A
  System.out.println(unicodeChar); // Natija: A
  ```

## Mantiqiy tur (Boolean Type)
### `boolean`
- **Ta’rif**: Faqat ikkita qiymat qabul qila oladi: `true` yoki `false`. Taxminan 1 bit xotira ishlatadi (amalda xotira hajmi JVM’ga bog‘liq).
- **Standart qiymat**: `false`
- **Foydalanish**: Shartli ifodalar, tsikllar va mantiqiy tekshiruvlar uchun ishlatiladi.
- **Misol**:
  ```java
  boolean isActive = true;
  boolean isEmpty = false;
  System.out.println(isActive); // Natija: true
  ```

## Primitiv turlarning xususiyatlari
- **Xotira samaradorligi**: Primitiv turlar ob’ektlarga nisbatan kamroq xotira ishlatadi, chunki ular to‘g‘ridan-to‘g‘ri qiymatlarni saqlaydi.
- **Standart qiymatlar**: Agar o‘zgaruvchi sinf ichida e’lon qilinsa va qiymat berilmasa, u avtomatik ravishda standart qiymatga ega bo‘ladi (masalan, `int` uchun 0, `boolean` uchun `false`).
- **Qattiq tipizatsiya**: Har bir o‘zgaruvchi turi oldindan aniqlanadi va unga boshqa turdagi qiymat yuklash xatolarga olib keladi (agar avtomatik yoki qo‘lda konvertatsiya qilinmasa).
- **Konvertatsiya**:
  - **Avtomatik (Widening)**: Kichik turdan katta turga o‘tish (masalan, `int` → `long`).
    ```java
    int small = 10;
    long big = small; // Avtomatik konvertatsiya
    ```
  - **Qo‘lda (Narrowing)**: Katta turdan kichik turga o‘tish, lekin bu ma'lumot yo‘qotilishiga olib kelishi mumkin.
    ```java
    double d = 10.5;
    int i = (int) d; // Qo‘lda konvertatsiya, i = 10
    ```

## Primitiv turlarni ishlatish bo‘yicha maslahatlar
1. **To‘g‘ri tur tanlash**: Xotirani tejash uchun kerakli eng kichik turdan foydalaning (masalan, kichik sonlar uchun `byte` yoki `short`).
2. **Aniqlik**: O‘nlik sonlar bilan ishlaganda, agar yuqori aniqlik kerak bo‘lsa, `double` ni tanlang.
3. **Unicode**: `char` turi bilan ishlaganda Unicode belgilaridan foydalanib, turli tillardagi belgilarni qo‘llash mumkin.
4. **Mantiqiy shartlar**: `boolean` turini shartli ifodalar va tsikllarda samarali ishlating.

## Misol: Barcha primitiv turlarni bir dasturda ko‘rsatish
```java
public class PrimitiveTypesExample {
    public static void main(String[] args) {
        // Butun son turlari
        byte b = 100;
        short s = 1000;
        int i = 100000;
        long l = 1000000000L;

        // O‘nlik son turlari
        float f = 3.14f;
        double d = 3.14159265359;

        // Belgi turi
        char c = 'A';

        // Mantiqiy tur
        boolean bool = true;

        // Natijalarni chop etish
        System.out.println("byte: " + b);
        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        System.out.println("char: " + c);
        System.out.println("boolean: " + bool);
    }
}
```

**Natija**:
```
byte: 100
short: 1000
int: 100000
long: 1000000000
float: 3.14
double: 3.14159265359
char: A
boolean: true
```
