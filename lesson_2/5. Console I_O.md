# Java’dagi Consol

## Java’dagi Consol nima?
Java’dagi **konsol** – bu dastur natijalarini matn shaklida chiqarish (masalan, terminal yoki command line oynasida) yoki foydalanuvchidan matnli kiritmalarni olish uchun ishlatiladigan standart kirish/chiqish (input/output) mexanizmi. Konsol dasturlashda sinov (debugging), oddiy dasturlarni ishlatish yoki matnli interfeysli dasturlar yaratish uchun qo‘llaniladi.

Java’da konsol bilan ishlash uchun bir nechta class'lar va metodlar mavjud, lekin eng muhimi `System` class'i va unga tegishli oqimlar: `System.out` (chiqish uchun), `System.in` (kirish uchun) va `System.err` (xato xabarlari uchun). Bundan tashqari, `java.util.Scanner` class'i konsoldan ma'lumotlarni o‘qishda keng tarqalgan.

## Konsol bilan ishlashning asosiy class'lari va metodlari

### 1. `System.out` (Chiqish uchun)
- **Ta’rif**: `System.out` – bu `PrintStream` class'ining ob’ekti bo‘lib, konsolga matn chiqarish uchun ishlatiladi.
- **Asosiy metodlar**:
  - `print(String s)`: Matnni konsolga chiqaradi, lekin yangi qatorga o‘tmaydi.
  - `println(String s)`: Matnni konsolga chiqaradi va oxirida yangi qator qo‘shadi.
  - `printf(String format, Object... args)`: Formatlangan matnni chiqarish uchun ishlatiladi (C tilidagi `printf` ga o‘xshash).
- **Misol**:
  ```java
  public class ConsoleOutputExample {
      public static void main(String[] args) {
          System.out.print("Salom, "); // Yangi qatorga o‘tmaydi
          System.out.println("Dunyo!"); // Yangi qatorga o‘tadi
          System.out.printf("Yosh: %d, Ism: %s%n", 25, "Ali"); // Formatlangan chiqish
      }
  }
  ```
  **Natija**:
  ```
  Salom, Dunyo!
  Yosh: 25, Ism: Ali
  ```

### 2. `System.in` (Kirish uchun)
- **Ta’rif**: `System.in` – bu `InputStream` class'ining ob’ekti bo‘lib, konsoldan ma'lumotlarni o‘qish uchun ishlatiladi. Odatda, `Scanner` class'i bilan birgalikda ishlatiladi.
- **Misol** (Scanner bilan):
  ```java
  import java.util.Scanner;

  public class ConsoleInputExample {
      public static void main(String[] args) {
          Scanner scanner = new Scanner(System.in);
          System.out.print("Ismingizni kiriting: ");
          String name = scanner.nextLine(); // Satr o‘qish
          System.out.print("Yoshingizni kiriting: ");
          int age = scanner.nextInt(); // Butun son o‘qish
          System.out.printf("Salom, %s! Siz %d yoshdasiz.%n", name, age);
          scanner.close(); // Resursni yopish
      }
  }
  ```
  **Natija** (foydalanuvchi kiritgan ma'lumotga qarab):
  ```
  Ismingizni kiriting: Ali
  Yoshingizni kiriting: 25
  Salom, Ali! Siz 25 yoshdasiz.
  ```

### 3. `System.err` (Xato xabarlari uchun)
- **Ta’rif**: `System.err` – bu xato xabarlarini chiqarish uchun ishlatiladigan `PrintStream` ob’ekti. Odatda qizil rangda ko‘rinadi (agar konsol ranglarni qo‘llab-quvvatlasa).
- **Misol**:
  ```java
  public class ConsoleErrorExample {
      public static void main(String[] args) {
          System.err.println("Xato: Noto‘g‘ri ma'lumot kiritildi!");
      }
  }
  ```
  **Natija**:
  ```
  Xato: Noto‘g‘ri ma'lumot kiritildi! (qizil rangda, agar konsol qo‘llab-quvvatlasa)
  ```

### 4. `java.util.Scanner` class'i
- **Ta’rif**: `Scanner` class'i konsoldan turli ma'lumot turlarini (satr, butun son, o‘nlik son va hokazo) o‘qish uchun ishlatiladi. `System.in` bilan birgalikda ishlaydi.
- **Asosiy metodlar**:
  - `nextLine()`: Butun qatorni satr sifatida o‘qiydi.
  - `nextInt()`: Butun son o‘qiydi.
  - `nextDouble()`: O‘nlik son o‘qiydi.
  - `next()`: Keyingi tokenni (bo‘shliqgacha bo‘lgan so‘zni) o‘qiydi.
  - `hasNext()`: Keyingi ma'lumot mavjudligini tekshiradi.
- **Eslatma**: `Scanner` ishlatilganda, foydalanish tugagach, `close()` metodi bilan yopish tavsiya etiladi.

## Konsol bilan ishlashning afzalliklari va kamchiliklari

### Afzalliklari:
- **Oddiylik**: Konsol oddiy va tez sinovlar uchun qulay.
- **Universal**: Har qanday platformada ishlaydi (Windows, Linux, macOS).
- **O‘quv maqsadlari**: Yangi boshlovchilar uchun dasturlashni o‘rganishda foydali.
- **Debugging**: Dastur natijalarini tezda ko‘rish imkonini beradi.

### Kamchiliklari:
- **Cheklangan funksionallik**: Konsol faqat matnli interfeysni qo‘llab-quvvatlaydi, grafik interfeys (GUI) yaratib bo‘lmaydi.
- **Foydalanuvchi tajribasi**: Zamonaviy dasturlarda konsol o‘rniga GUI yoki veb-interfeyslar afzal ko‘riladi.

## Konsol bilan ishlashda xatolar va ulardan saqlanish
1. **Noto‘g‘ri kiritish turi**:
   - **Muammo**: Agar foydalanuvchi `nextInt()` o‘rniga harf kiritsa, `InputMismatchException` yuzaga keladi Bondi
  - **Yechim**: `try-catch` blokidan foydalaning.
  - **Misol**:
    ```java
    Scanner scanner = new Scanner(System.in);
    try {
        System.out.print("Yoshingizni kiriting: ");
        int age = scanner.nextInt();
        System.out.println("Yosh: " + age);
    } catch (java.util.InputMismatchException e) {
        System.out.println("Xato: Faqat raqam kiriting!");
    }
    scanner.close();
    ```

2. **Scanner resursini yopmaslik**:
   - **Muammo**: `Scanner` ob’ekti yopilmasa, xotira oqishi (memory leak) yuzaga kelishi mumkin.
   - **Yechim**: Har doim `scanner.close()` ni chaqiring.

3. **Qator oxiridagi muammolar**:
   - **Muammo**: `nextInt()` yoki `nextDouble()` dan keyin `nextLine()` ishlatilsa, qator oxiridagi `\n` belgisi o‘qilib ketishi mumkin.
   - **Yechim**: `nextLine()` dan oldin qo‘shimcha `scanner.nextLine()` chaqiring.
   - **Misol**:
     ```java
     Scanner scanner = new Scanner(System.in);
     System.out.print("Yoshingizni kiriting: ");
     int age = scanner.nextInt();
     scanner.nextLine(); // Qator oxirini tozalash
     System.out.print("Ismingizni kiriting: ");
     String name = scanner.nextLine();
     System.out.printf("Salom, %s! Siz %d yoshdasiz.%n", name, age);
     scanner.close();
     ```

## Amaliy Misol: Konsol orqali kalkulyator dasturi
Quyidagi dastur konsoldan ikkita son va amalni oladi va natijani chiqaradi:

```java
import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Birinchi sonni kiriting: ");
        double num1 = scanner.nextDouble();

        System.out.print("Ikkinchi sonni kiriting: ");
        double num2 = scanner.nextDouble();

        System.out.print("Amalni kiriting (+, -, *, /): ");
        char operator = scanner.next().charAt(0);

        double result;
        switch (operator) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    result = num1 / num2;
                } else {
                    System.err.println("Xato: Nolga bo‘lish mumkin emas!");
                    scanner.close();
                    return;
                }
                break;
            default:
                System.err.println("Xato: Noto‘g‘ri amal kiritildi!");
                scanner.close();
                return;
        }

        System.out.printf("Natija: %.2f%n", result);
        scanner.close();
    }
}
```

**Natija** (foydalanuvchi kiritgan ma'lumotga qarab):
```
Birinchi sonni kiriting: 10.5
Ikkinchi sonni kiriting: 5
Amalni kiriting (+, -, *, /): *
Natija: 52.50
```

## Qo‘shimcha Maslahatlar
- **Formatlash**: `printf` yordamida chiqishni chiroyli formatlash mumkin (masalan, o‘nlik sonlar uchun `%.2f`).
- **Konsol muhitlari**: Java konsoli IDE’larda (IntelliJ IDEA, Eclipse) yoki terminalda (Command Prompt, Bash) ishlaydi.
- **Kengaytirilgan konsol**: Zamonaviy dasturlarda konsol o‘rniga `java.io.Console` class'i ishlatilishi mumkin, lekin u faqat haqiqiy konsol muhitida ishlaydi (IDE’da emas).
- **Alternativlar**: Konsol o‘rniga GUI (Swing, JavaFX) yoki veb-interfeyslar ishlatilishi mumkin.
