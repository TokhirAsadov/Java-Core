# Java’dagi Scanner class'i

## Scanner class'i nima?
`java.util.Scanner` class'i Java’da konsoldan, fayllardan yoki boshqa kirish manbalaridan (masalan, satrlar) ma’lumotlarni o‘qish uchun ishlatiladi. U turli ma’lumot turlarini (satr, butun son, o‘nlik son, belgi va hokazo) osonlik bilan o‘qish imkonini beradi. `Scanner` class'i `System.in` (konsol kirishi), fayl oqimlari (`FileInputStream`) yoki satrlar (`String`) kabi kirish manbalari bilan ishlaydi.

**Asosiy xususiyatlari**:
- `Scanner` matnni tokenlarga (bo‘shliq yoki maxsus belgilar bilan ajratilgan qismlarga) bo‘ladi va ularni kerakli ma’lumot turiga aylantiradi.
- Foydalanuvchidan interaktiv ravishda ma’lumot olish uchun qulay.
- Turli ma’lumot turlarini o‘qish uchun maxsus metodlar (masalan, `nextInt()`, `nextDouble()`) taqdim etadi.

## Scanner class'ini Ishlatish
`Scanner` class'ini ishlatish uchun avval `java.util.Scanner` paketini import qilish kerak. Keyin `Scanner` ob’ekti yaratiladi, unga kirish manbai (masalan, `System.in`) beriladi.

**Oddiy misol**:
```java
import java.util.Scanner;

public class ScannerExample {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in); // Konsoldan o‘qish uchun
        System.out.print("Ismingizni kiriting: ");
        String name = scanner.nextLine();
        System.out.print("Yoshingizni kiriting: ");
        int age = scanner.nextInt();
        System.out.printf("Salom, %s! Siz %d yoshdasiz.%n", name, age);
        scanner.close();
    }
}
```

**Natija** (foydalanuvchi kiritgan ma’lumotga qarab):
```
Ismingizni kiriting: Ali
Yoshingizni kiriting: 25
Salom, Ali! Siz 25 yoshdasiz.
```

## Asosiy Metodlar
`Scanner` class'i turli ma’lumot turlarini o‘qish va kirishni boshqarish uchun bir qator metodlarni taqdim etadi. Quyida eng muhim metodlar keltiriladi:

### 1. Ma’lumot o‘qish metodlari
- **`nextLine()`**: Butun qatorni satr (`String`) sifatida o‘qiydi.
- **`next()`**: Keyingi tokenni (bo‘shliqgacha bo‘lgan so‘zni) satr sifatida o‘qiydi.
- **`nextInt()`**: Keyingi tokenni butun son (`int`) sifatida o‘qiydi.
- **`nextDouble()`**: Keyingi tokenni o‘nlik son (`double`) sifatida o‘qiydi.
- **`nextFloat()`**: Keyingi tokenni o‘nlik son (`float`) sifatida o‘qiydi.
- **`nextLong()`**: Keyingi tokenni uzun butun son (`long`) sifatida o‘qiydi.
- **`nextShort()`**: Keyingi tokenni qisqa butun son (`short`) sifatida o‘qiydi.
- **`nextByte()`**: Keyingi tokenni bayt (`byte`) sifatida o‘qiydi.
- **`nextBoolean()`**: Keyingi tokenni mantiqiy qiymat (`true` yoki `false`) sifatida o‘qiydi.
- **`next().charAt(0)`**: Keyingi tokenning birinchi belgisini (`char`) olish uchun ishlatiladi.

**Misol**:
```java
Scanner scanner = new Scanner(System.in);
System.out.print("Satr kiriting: ");
String text = scanner.nextLine(); // "Salom Dunyo"
System.out.print("Butun son kiriting: ");
int number = scanner.nextInt(); // 42
System.out.print("O‘nlik son kiriting: ");
double decimal = scanner.nextDouble(); // 3.14
System.out.print("Belgi kiriting: ");
char ch = scanner.next().charAt(0); // 'A'
scanner.close();
```

### 2. Ma’lumot mavjudligini tekshirish metodlari
- **`hasNext()`**: Keyingi token mavjudligini tekshiradi (umumiy).
- **`hasNextInt()`**: Keyingi token butun son ekanligini tekshiradi.
- **`hasNextDouble()`**: Keyingi token o‘nlik son ekanligini tekshiradi.
- **`hasNextLine()`**: Keyingi qator mavjudligini tekshiradi.
- **Boshqa o‘xshash metodlar**: `hasNextFloat()`, `hasNextLong()`, `hasNextByte()`, `hasNextBoolean()`.

**Misol**:
```java
Scanner scanner = new Scanner(System.in);
if (scanner.hasNextInt()) {
    int number = scanner.nextInt();
    System.out.println("Kiritilgan son: " + number);
} else {
    System.out.println("Butun son kiritilmadi!");
}
scanner.close();
```

### 3. Boshqa muhim metodlar
- **`close()`**: `Scanner` ob’ektini yopadi va resurslarni bo‘shatadi.
- **`useDelimiter(String pattern)`**: Tokenlarni ajratish uchun maxsus ajratuvchi (delimiter) o‘rnatadi (standart bo‘shliq yoki qator oxiri).
- **`skip(String pattern)`**: Belgilangan naqshga mos keluvchi ma’lumotlarni o‘tkazib yuboradi.
- **`reset()`**: Scanner’ni standart sozlamalariga qaytaradi.

**Misol (Delimiter)**:
```java
Scanner scanner = new Scanner("Ali,Vali,Said");
scanner.useDelimiter(",");
while (scanner.hasNext()) {
    System.out.println(scanner.next()); // Ali, Vali, Said alohida chiqadi
}
scanner.close();
```

## Scanner’ni turli manbalardan foydalanish
`Scanner` nafaqat konsoldan (`System.in`), balki boshqa manbalardan ham ma’lumot o‘qishi mumkin:

1. **Fayldan o‘qish**:
```java
import java.util.Scanner;
import java.io.File;

public class FileScannerExample {
    public static void main(String[] args) throws Exception {
        File file = new File("data.txt");
        Scanner scanner = new Scanner(file);
        while (scanner.hasNextLine()) {
            System.out.println(scanner.nextLine());
        }
        scanner.close();
    }
}
```

2. **Satr (String) dan o‘qish**:
```java
Scanner scanner = new Scanner("10 20 30");
int sum = 0;
while (scanner.hasNextInt()) {
    sum += scanner.nextInt();
}
System.out.println("Yig‘indi: " + sum); // Natija: 60
scanner.close();
```

## Xatolar va Ulardan Saqlanish

1. **Noto‘g‘ri kiritish turi (`InputMismatchException`)**:
   - **Muammo**: Foydalanuvchi `nextInt()` uchun raqam o‘rniga harf kiritsa, `InputMismatchException` yuzaga keladi.
   - **Yechim**: `try-catch` blokidan foydalaning yoki `hasNextInt()` bilan tekshiring.
   - **Misol**:
     ```java
     Scanner scanner = new Scanner(System.in);
     try {
         System.out.print("Butun son kiriting: ");
         int number = scanner.nextInt();
         System.out.println("Son: " + number);
     } catch (java.util.InputMismatchException e) {
         System.out.println("Xato: Faqat butun son kiriting!");
     }
     scanner.close();
     ```

2. **Qator oxiridagi muammo**:
   - **Muammo**: `nextInt()` yoki `nextDouble()` dan keyin `nextLine()` ishlatilsa, qator oxiridagi `\n` belgisi o‘qilib, bo‘sh satr qaytarilishi mumkin.
   - **Yechim**: `nextLine()` dan oldin qo‘shimcha `scanner.nextLine()` chaqiring.
   - **Misol**:
     ```java
     Scanner scanner = new Scanner(System.in);
     System.out.print("Yoshingizni kiriting: ");
     int age = scanner.nextInt();
     scanner.nextLine(); // Qator oxirini tozalash
     System.out.print("Ismingizni kiriting: ");
     String name = scanner.nextLine();
     System.out.printf("Salom, %s! Siz %d yoshdasiz.%n", name, age);
     scanner.close();
     ```

3. **Resurslarni yopmaslik**:
   - **Muammo**: `Scanner` yopilmasa, xotira oqishi (memory leak) yuzaga kelishi mumkin.
   - **Yechim**: Har doim `scanner.close()` ni chaqiring, ayniqsa fayllar bilan ishlaganda.

4. **Fayl topilmadi (`FileNotFoundException`)**:
   - **Muammo**: Fayldan o‘qishda fayl topilmasa, xato yuzaga keladi.
   - **Yechim**: `try-catch` bilan fayl mavjudligini tekshiring.
   - **Misol**:
     ```java
     try {
         Scanner scanner = new Scanner(new File("data.txt"));
         while (scanner.hasNextLine()) {
             System.out.println(scanner.nextLine());
         }
         scanner.close();
     } catch (java.io.FileNotFoundException e) {
         System.out.println("Xato: Fayl topilmadi!");
     }
     ```

## Amaliy Misol: Konsol Kalkulyatori
Quyidagi dastur `Scanner` yordamida foydalanuvchidan ikkita son va amalni oladi va natijani chiqaradi:

```java
import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Birinchi sonni kiriting: ");
        if (!scanner.hasNextDouble()) {
            System.out.println("Xato: O‘nlik son kiriting!");
            scanner.close();
            return;
        }
        double num1 = scanner.nextDouble();

        System.out.print("Ikkinchi sonni kiriting: ");
        if (!scanner.hasNextDouble()) {
            System.out.println("Xato: O‘nlik son kiriting!");
            scanner.close();
            return;
        }
        double num2 = scanner.nextDouble();

        scanner.nextLine(); // Qator oxirini tozalash
        System.out.print("Amalni kiriting (+, -, *, /): ");
        String operator = scanner.nextLine();

        double result;
        switch (operator.charAt(0)) {
            case '+':
                result = num1 + num2;
                break;
            case '-':
                result = num1 - num2;
                break;
            case '*':
                result = num1 * num2;
                break;
            case '/':
                if (num2 != 0) {
                    result = num1 / num2;
                } else {
                    System.out.println("Xato: Nolga bo‘lish mumkin emas!");
                    scanner.close();
                    return;
                }
                break;
            default:
                System.out.println("Xato: Noto‘g‘ri amal kiritildi!");
                scanner.close();
                return;
        }

        System.out.printf("Natija: %.2f%n", result);
        scanner.close();
    }
}
```

**Natija**:
```
Birinchi sonni kiriting: 10.5
Ikkinchi sonni kiriting: 5
Amalni kiriting (+, -, *, /): *
Natija: 52.50
```

## Qo‘shimcha Maslahatlar
- **Ma’lumot turini tekshirish**: Har doim `hasNextInt()`, `hasNextDouble()` kabi metodlardan foydalanib, kiritilgan ma’lumotning to‘g‘ri ekanligini tekshiring.
- **Resurslarni boshqarish**: `Scanner` ob’ektini ishlatib bo‘lgach, `close()` metodi bilan yoping.
- **Delimiter sozlash**: Maxsus ajratuvchilar (masalan, vergul yoki bo‘shliq) bilan ishlash uchun `useDelimiter()` dan foydalaning.
- **Katta hajmdagi ma’lumotlar**: Agar katta fayllarni o‘qish kerak bo‘lsa, `BufferedReader` class'ini ham ko‘rib chiqing, chunki u ko‘proq samarali bo‘lishi mumkin.
- **Konsol cheklovlari**: `Scanner` IDE konsollarida yaxshi ishlaydi, lekin `java.io.Console` class'i faqat haqiqiy terminal muhitida ishlaydi.
